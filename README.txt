Welcome to the Bootcamp Repository!!

Use "javac Message.java"
and "java Message"
to compile and run the Message program 

If you add any files you can Use
"git add <files>" to add any specific files OR
"git add ." to add all changes

Finally, use:
"git fetch" to refresh the repository
"git pull" to update any changes you don't have yet
"git commit -m "Added files"" to commit 
and "git push" to push your files to the repository
